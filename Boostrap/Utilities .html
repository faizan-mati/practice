üì¶ Overflow Utilities

Control how content overflows its container

Overflow Classes:

  • .overflow-auto - Auto scrollbars
  • .overflow-hidden - Hide overflow
  • .overflow-visible - Show overflow
  • .overflow-scroll - Always show scrollbars
Directional:
  • .overflow-x-* - Horizontal only
  • .overflow-y-* - Vertical only

Live Examples:

overflow-hidden

This is a very long text that would normally overflow the container, but it's being hidden due to overflow-hidden class. You won't see scrollbars here.

overflow-auto

This is a very long text that will show scrollbars when needed. The overflow-auto class adds scrollbars only when content overflows.

Real-world Use Cases:
  • Image Containers: Use overflow-hidden to crop images
  • Chat Windows: Use overflow-y-auto for scrollable message lists
  • Code Blocks: Use overflow-x-auto for horizontal scrolling
  • Modal Bodies: Use overflow-auto for long content

üìè Spacing Utilities

Control margin and padding with spacing utilities

Spacing Syntax:

{property}{sides}-{size}

Properties:
  • m - margin
  • p - padding
Sides:
  • t - top
  • b - bottom
  • s - start (left)
  • e - end (right)
  • x - horizontal
  • y - vertical
  • (blank) - all sides
Sizes:

0, 1, 2, 3, 4, 5, auto
