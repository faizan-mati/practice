üöÄ Getting Started with Bootstrap 5 CDN

What is CDN?

CDN (Content Delivery Network) allows you to use Bootstrap without downloading files. Just include these links in your HTML:

CSS CDN (in <head>):
<link href="https://cdnjs.cloudflare.com/ajax/libs/bootstrap/5.3.2/css/bootstrap.min.css" rel="stylesheet">

JavaScript CDN (before </body>):
<script src="https://cdnjs.cloudflare.com/ajax/libs/bootstrap/5.3.2/js/bootstrap.bundle.min.js"></script>
Benefits: No downloads, always updated, fast loading, works offline after first load
