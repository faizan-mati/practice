Key Points Summary

Cards:

  • card: Main container class
  • card-img-top: Image at top spanning full width
  • card-body: Main content area
  • card-header/footer: Optional sections
  • card-title: Title styling
  • card-text: Body text styling

Badges:

  • badge: Base badge class
  • bg-secondary: Gray/neutral color
  • bg-*: Various color options
  • rounded-pill: Pill-shaped badges
  • Perfect for tags, notifications, status indicators
  • Can be combined with other Bootstrap components
