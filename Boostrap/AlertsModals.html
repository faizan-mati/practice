Bootstrap Alerts

Alerts provide contextual feedback messages for typical user actions with flexible alert messages.

1. Basic Alerts

Primary alert - Important information
Secondary alert - General information
Success alert - Operation completed successfully
Danger alert - Critical error occurred
Warning alert - Proceed with caution
Info alert - Additional information
Light alert - Subtle information
Dark alert - Strong contrast information
<div class="alert alert-success" role="alert"> <i class="fas fa-check-circle"></i> Success alert - Operation completed successfully </div>

2. Dismissible Alerts

Warning! This is a dismissible alert. Click the × to close.
Did you know? You can dismiss this alert and it will fade out smoothly.
<div class="alert alert-warning alert-dismissible fade show" role="alert"> <strong>Warning!</strong> This is a dismissible alert. <button type="button" class="btn-close" data-bs-dismiss="alert"></button> </div>

Getting Started
  • Click on tooltips to see information
  • Use modals for detailed actions
  • Pay attention to alert messages
Keyboard Shortcuts
  • Esc - Close modals
  • Tab - Navigate elements
  • Enter - Activate buttons
Pro Tip: All components are fully accessible and keyboard navigable!
Key Takeaways:
  • • Alerts: Use for user feedback and notifications
  • • Modals: Perfect for forms, confirmations, and detailed content
  • • Tooltips: Provide contextual help without cluttering the UI
  • • data-bs-toggle: The key attribute for Bootstrap JavaScript components
Quick Reference:
