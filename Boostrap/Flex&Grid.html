10. Responsive Flexbox Utilities

Responsive Patterns:
  • .d-flex .d-md-block - flex on mobile, block on desktop
  • .justify-content-center .justify-content-md-start
  • .flex-column .flex-md-row - stack on mobile, row on desktop
  • .align-items-center .align-items-md-end
Breakpoint Reference:
  • xs: <576px (no infix needed)
  • sm: ≥576px (.d-sm-*)
  • md: ≥768px (.d-md-*)
  • lg: ≥992px (.d-lg-*)
  • xl: ≥1200px (.d-xl-*)
  • xxl: ≥1400px (.d-xxl-*)

12. Best Practices & Tips

✅ Best Practices:
  • Use d-flex for one-dimensional layouts
  • Use d-grid for two-dimensional layouts
  • Combine with responsive utilities for mobile-first design
  • Use flex-fill for equal-width columns
  • Use align-items-center for vertical centering
⚠️ Common Mistakes:
  • Forgetting to add d-flex before flexbox utilities
  • Not understanding the difference between main axis and cross axis
  • Overusing flexbox when CSS Grid would be better
  • Not testing responsive behavior across breakpoints
