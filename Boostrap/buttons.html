Basic Button Classes

Standard Buttons (.btn)
<button class="btn btn-primary">Primary</button> <button class="btn btn-success">Success</button> <button class="btn btn-danger">Danger</button>
Real-World Use: Primary for main actions (Save, Submit), Success for confirmations, Danger for deletions, Warning for important notices.
Outline Buttons (.btn-outline-*)
<button class="btn btn-outline-primary">Outline Primary</button> <button class="btn btn-outline-success">Outline Success</button>
Real-World Use: Secondary actions, ghost buttons for minimalist designs, call-to-action buttons that need less visual weight.

Button Sizes

Size Variations
<button class="btn btn-primary btn-lg">Large</button> <button class="btn btn-primary">Default</button> <button class="btn btn-primary btn-sm">Small</button>
Real-World Use: Large for hero sections, small for compact interfaces like tables or cards.
Block Buttons
<div class="d-grid gap-2"> <button class="btn btn-primary">Block Button</button> </div>
Real-World Use: Mobile interfaces, form submissions, call-to-action sections.

Button States

Interactive States
<button class="btn btn-primary active">Active</button> <button class="btn btn-primary" disabled>Disabled</button> <button class="btn btn-primary" data-bs-toggle="button">Toggle</button>
Real-World Use: Active for current page/selection, disabled during loading/processing, toggle for settings switches.

Button Groups (.btn-group)

Basic Button Group
<div class="btn-group" role="group"> <button type="button" class="btn btn-primary">Left</button> <button type="button" class="btn btn-primary">Middle</button> <button type="button" class="btn btn-primary">Right</button> </div>
Real-World Use: Navigation tabs, view toggles (List/Grid), text formatting tools (Bold/Italic/Underline).

Vertical Layout
