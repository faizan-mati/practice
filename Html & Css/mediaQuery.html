CSS Media Queries Complete Guide

1. min-width Media Queries (Mobile-First Approach)

Min-width queries apply styles when the viewport is at least the specified width. This follows the mobile-first responsive design principle.

Live Demo - Resize your browser window!

This box changes color and size based on screen width
/* Mobile First - Base styles */ .responsive-box { font-size: 14px; background: linear-gradient(45deg, #ff6b6b, #ee5a24); } /* Small devices (‚â•576px) */ @media (min-width: 576px) { .responsive-box { font-size: 16px; background: linear-gradient(45deg, #feca57, #ff9ff3); } } /* Medium devices (‚â•768px) */ @media (min-width: 768px) { .responsive-box { font-size: 18px; background: linear-gradient(45deg, #48dbfb, #0abde3); } } /* Large devices (‚â•992px) */ @media (min-width: 992px) { .responsive-box { font-size: 20px; background: linear-gradient(45deg, #1dd1a1, #10ac84); } } /* Extra large devices (‚â•1200px) */ @media (min-width: 1200px) { .responsive-box { font-size: 22px; background: linear-gradient(45deg, #a55eea, #8b5cf6); } }

2. max-width Media Queries (Desktop-First Approach)

Max-width queries apply styles when the viewport is at most the specified width. Used in desktop-first responsive design.

3. Orientation Media Queries

Orientation queries detect whether the device is in portrait (height > width) or landscape (width > height) mode.

Live Demo - Rotate your device or resize window!

Orientation Detection:
/* Portrait orientation (height > width) */ @media (orientation: portrait) { .element { background: linear-gradient(135deg, #ff6b6b, #ee5a24); /* Styles for tall, narrow screens */ } } /* Landscape orientation (width > height) */ @media (orientation: landscape) { .element { background: linear-gradient(135deg, #1dd1a1, #10ac84); /* Styles for wide screens */ } } /* Real-world example: Navigation */ @media (orientation: landscape) and (max-height: 500px) { .navigation { height: 40px; /* Shorter nav for landscape phones */ } }

4. prefers-color-scheme (Dark/Light Mode)

This query detects the user's system-level preference for dark or light color schemes.

Live Demo - Check your system's dark/light mode setting!

Theme Detection:
/* Light mode (default) */ .theme-element { background: #ffffff; color: #333333; border: 1px solid #e0e0e0; } /* Dark mode */ @media (prefers-color-scheme: dark) { .theme-element { background: #2d2d2d; color: #ffffff; border: 1px solid #555555; } } /* Explicitly light mode */ @media (prefers-color-scheme: light) { .theme-element { background: #fafafa; color: #222222; } } /* No preference - fallback styles */ @media (prefers-color-scheme: no-preference) { .theme-element { /* Fallback styles */ } }

5. Combined Media Queries

You can combine multiple conditions using and, or, and not operators.

Live Demo - Complex Conditions

Combined Query Result:
/* AND operator - All conditions must be true */ @media (min-width: 768px) and (orientation: landscape) { .element { /* Tablets/desktops in landscape */ } } /* Multiple conditions */ @media (max-width: 768px) and (orientation: landscape) and (prefers-color-scheme: dark) { .element { /* Mobile landscape + dark mode */ background: #1a1a1a; color: #ffffff; } } /* OR operator using comma separation */ @media (max-width: 600px), (orientation: portrait) { .element { /* Small screens OR portrait mode */ } } /* NOT operator */ @media not screen and (max-width: 600px) { .element { /* All non-screen media or screens wider than 600px */ } }
