✨ Smooth Scrolling Behavior

The navigation buttons above use smooth scrolling. Notice how the page scrolls smoothly instead of jumping.

/* Enable smooth scrolling for entire page */ html { scroll-behavior: smooth; /* Makes all scrolling smooth */ } /* JavaScript smooth scrolling */ element.scrollIntoView({ behavior: 'smooth', block: 'start' });

📚 Complete Property Reference

/* scroll-snap-type values */ scroll-snap-type: none; /* No snapping */ scroll-snap-type: x mandatory; /* Must snap horizontally */ scroll-snap-type: y mandatory; /* Must snap vertically */ scroll-snap-type: x proximity; /* Optional horizontal snap */ scroll-snap-type: y proximity; /* Optional vertical snap */ scroll-snap-type: both mandatory; /* Both directions required */ /* scroll-snap-align values */ scroll-snap-align: none; /* No alignment */ scroll-snap-align: start; /* Align to start edge */ scroll-snap-align: end; /* Align to end edge */ scroll-snap-align: center; /* Align to center */ /* scroll-behavior values */ scroll-behavior: auto; /* Instant scrolling */ scroll-behavior: smooth; /* Smooth scrolling */ /* Additional properties */ scroll-snap-stop: normal; /* Allow passing through */ scroll-snap-stop: always; /* Force stop at each snap point */
