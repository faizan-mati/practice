1. SRCSET Attribute

Purpose: Provides multiple image sources for different screen densities and sizes, allowing browsers to choose the most appropriate image.

Basic Syntax:

<img src="default.jpg" srcset="small.jpg 480w, medium.jpg 768w, large.jpg 1200w" alt="Responsive image">

Density Descriptors (for Retina displays):

<img src="image.jpg" srcset="image.jpg 1x, image@2x.jpg 2x, image@3x.jpg 3x" alt="High DPI image">

2. SIZES Attribute

Purpose: Tells the browser how much space the image will occupy in the layout at different viewport sizes.

Basic Syntax:

<img src="default.jpg" srcset="small.jpg 480w, medium.jpg 768w, large.jpg 1200w" sizes="(max-width: 480px) 100vw, (max-width: 768px) 50vw, 33vw" alt="Responsive image">

Complex Layout Example:

sizes="(max-width: 320px) 280px, (max-width: 640px) calc(100vw - 40px), (max-width: 1024px) calc(50vw - 20px), 400px"

Live Demo - Gallery Layout:

The sizes attribute helps the browser choose the right image based on layout space, not just viewport size.

Important: Without the sizes attribute, browsers assume the image will be 100vw (full viewport width), which may lead to downloading unnecessarily large images.

3. OBJECT-FIT Property

Purpose: Controls how an image should be resized to fit its container, similar to background-size for background images.

All Object-Fit Values:

.image { width: 200px; height: 150px; object-fit: cover; /* Crops to fill container */ object-fit: contain; /* Scales to fit inside container */ object-fit: fill; /* Stretches to fill (default) */ object-fit: scale-down; /* Acts like contain or none, whichever is smaller */ object-fit: none; /* No scaling */ }
