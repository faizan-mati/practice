üîµ CSS clip-path Property

clip-path creates a clipping region that sets what part of an element should be shown. Parts outside the region are hidden.

circle()

clip-path: circle(50%);

ellipse()

clip-path: ellipse(40% 30%);

polygon() - Triangle

clip-path: polygon(50% 0%, 0% 100%, 100% 100%);

polygon() - Hexagon

clip-path: polygon(30% 0%, 70% 0%, 100% 50%, 70% 100%, 30% 100%, 0% 50%);

polygon() - Star

clip-path: polygon(50% 0%, 61% 35%, 98% 35%, 68% 57%, 79% 91%, 50% 70%, 21% 91%, 32% 57%, 2% 35%, 39% 35%);

inset()

clip-path: inset(20px 30px 40px 20px round 15px);

üé≠ CSS mask-image Property

mask-image sets the image that is used as a mask layer for an element. The mask defines which parts are visible (black = visible, transparent = hidden).

Linear Gradient Mask

mask-image: linear-gradient(to right, transparent, black, transparent);

Radial Gradient Mask

mask-image: radial-gradient(circle at center, black 40%, transparent 70%);

Repeating Pattern Mask

mask-image: repeating-linear-gradient(45deg, black 0px, black 20px, transparent 20px, transparent 40px);

clip-path Functions:

/* Basic shapes */ clip-path: circle(50px); /* Circle with 50px radius */ clip-path: ellipse(50px 100px); /* Ellipse with 50px horizontal, 100px vertical radius */ clip-path: inset(10px 20px 30px 40px); /* Rectangular inset from edges */ clip-path: polygon(0 0, 100% 0, 50% 100%); /* Triangle using coordinate points */ /* Advanced usage */ clip-path: circle(50% at 25% 75%); /* Circle at custom position */ clip-path: inset(10px round 15px); /* Inset with rounded corners */

mask-image Properties:

/* Gradient masks */ mask-image: linear-gradient(black, transparent); mask-image: radial-gradient(black 50%, transparent 70%); /* Image masks */ mask-image: url('mask.png'); /* Multiple masks */ mask-image: url('mask1.png'), linear-gradient(black, transparent); /* Mask properties */ mask-size: cover, contain, 100px 200px; mask-position: center, top left, 50% 25%; mask-repeat: no-repeat, repeat-x, repeat-y, repeat;

Browser Support & Performance:

  • clip-path: Excellent modern browser support, use vendor prefixes for older browsers
  • mask-image: Good support, always include -webkit- prefix
  • Performance: Both properties use GPU acceleration when possible
  • Accessibility: Ensure clipped/masked content doesn't hide important information
