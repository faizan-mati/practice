CSS Gradients Complete Guide

1. Linear Gradients - linear-gradient()

Linear gradients create smooth color transitions along a straight line.

Basic Syntax:

background: linear-gradient(/* direction */, /* color-stop1 */, /* color-stop2 */, ...);

Direction Options:

  • to right - Left to right (default)
  • to bottom - Top to bottom
  • to top - Bottom to top
  • to left - Right to left
  • 45deg - Angle in degrees
  • to top right - Diagonal direction

2. Radial Gradients - radial-gradient()

Radial gradients create smooth color transitions radiating from a center point.

Basic Syntax:

background: radial-gradient(/* shape */ /* size */ /* at position */, /* color-stop1 */, /* color-stop2 */, ...);

Shape Options:

  • circle - Perfect circle
  • ellipse - Oval shape (default)

Size Keywords:

  • closest-side - Ends at closest side
  • closest-corner - Ends at closest corner
  • farthest-side - Ends at farthest side (default)
  • farthest-corner - Ends at farthest corner

4. Advanced Techniques

Color Stops with Percentages:

/* Precise control over color positions */ background: linear-gradient(45deg, #ff6b6b 0%, #4ecdc4 50%, #45b7d1 100% );

Multiple Gradients:

/* Layering multiple gradients */ background: linear-gradient(45deg, rgba(255,0,0,0.5), transparent), radial-gradient(circle, blue, green);

Repeating Gradients:

/* Create patterns with repeating gradients */ background: repeating-linear-gradient( 45deg, #606dbc, #606dbc 35px, #465298 35px, #465298 70px );

5. Browser Compatibility & Best Practices

  • Modern gradients are supported in all current browsers
  • Use vendor prefixes for older browser support (-webkit-, -moz-, -o-)
  • Provide fallback colors for older browsers
  • Consider performance - complex gradients can impact rendering
  • Use rgba() colors for transparency effects
  • Test gradients across different screen sizes and devices

Fallback Example:

.gradient-element { background: #667eea; /* Fallback for old browsers */ background: -webkit-linear-gradient(45deg, #667eea, #764ba2); background: linear-gradient(45deg, #667eea, #764ba2); }
