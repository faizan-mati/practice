üé® CSS Filters

CSS filters allow you to apply graphical effects like blur, brightness, contrast, and more to elements. They're hardware-accelerated and perfect for creating stunning visual effects.

Blur Filter

The blur() function applies a Gaussian blur to elements. Great for creating depth, focus effects, and loading states.

Original

filter: none

Light Blur

filter: blur(2px)

Medium Blur

filter: blur(5px)

Heavy Blur

filter: blur(10px)

/* Blur Examples */ .light-blur { filter: blur(2px); } .medium-blur { filter: blur(5px); } .heavy-blur { filter: blur(10px); } /* Real-world use: Loading placeholder */ .loading-image { filter: blur(5px); transition: filter 0.3s ease; } .loading-image.loaded { filter: blur(0); }

üåü Real-World Applications

1. Image Hover Effects

Hover Over This Card

Uses filter: brightness(105%) saturate(110%) on hover for a subtle enhancement effect.

.card-hover-effect:hover { filter: brightness(105%) saturate(110%); transform: translateY(-5px); transition: all 0.3s ease; }

2. Image Overlay with Blend Modes

.image-overlay::after { content: ''; position: absolute; top: 0; left: 0; right: 0; bottom: 0; background: linear-gradient(45deg, rgba(102, 126, 234, 0.8), rgba(118, 75, 162, 0.8)); mix-blend-mode: multiply; opacity: 0; transition: opacity 0.3s ease; } .image-overlay:hover::after { opacity: 1; }

3. Glassmorphism Effect

Glass Morphism Card

Using backdrop-filter: blur(10px) combined with semi-transparent backgrounds creates this trendy glass effect.

.glass-effect { background: rgba(255, 255, 255, 0.1); backdrop-filter: blur(10px); border: 1px solid rgba(255, 255, 255, 0.2); border-radius: 15px; }

4. Loading States

Use blur filters to create elegant loading placeholders:

/* Image loading state */ .image-placeholder { filter: blur(5px); transition: filter 0.5s ease; } .image-placeholder.loaded { filter: blur(0); } /* Skeleton loading with grayscale */ .skeleton { filter: grayscale(100%) brightness(90%); animation: pulse 2s ease-in-out infinite alternate; } @keyframes pulse { from { filter: grayscale(100%) brightness(90%); } to { filter: grayscale(100%) brightness(110%); } }

5. Accessibility Features

/* High contrast mode */ @media (prefers-contrast: high) { .image { filter: contrast(150%); } } /* Reduced motion preference */ @media (prefers-reduced-motion: reduce) { .animated-filter { filter: none !important; transition: none !important; } } /* Dark mode adjustments */ @media (prefers-color-scheme: dark) { .bright-image { filter: brightness(80%); } }

üöÄ Performance Tips

  • Filters are GPU-accelerated but can impact performance on complex layouts
  • Use `will-change: filter` for elements that will be animated
  • Avoid applying filters to large areas or many elements simultaneously
  • Test on mobile devices as GPU performance varies significantly

Advanced Techniques

/* CSS Variables for dynamic filters */ :root { --blur-amount: 0px; --brightness: 100%; --saturation: 100%; } .dynamic-filter { filter: blur(var(--blur-amount)) brightness(var(--brightness)) saturate(var(--saturation)); transition: filter 0.3s ease; } /* JavaScript control */ // element.style.setProperty('--blur-amount', '5px'); // element.style.setProperty('--brightness', '120%'); /* Combining with transforms for complex effects */ .complex-effect { filter: blur(1px) brightness(110%) saturate(120%); transform: scale(1.05) rotate(1deg); mix-blend-mode: multiply; transition: all 0.4s cubic-bezier(0.4, 0, 0.2, 1); } .complex-effect:hover { filter: blur(0) brightness(100%) saturate(100%); transform: scale(1) rotate(0deg); }
