üéØ What Are Complex Grid Layouts?

Complex grid layouts solve common web design challenges by creating structured, flexible, and responsive layouts. The two most important patterns are:

A classic three-column layout with header and footer. The main content area expands while sidebars remain fixed width.

üíª CSS Implementation:

.masonry-container { display: grid; grid-template-columns: repeat(auto-fit, minmax(250px, 1fr)); grid-auto-rows: 10px; /* Small row units */ gap: 15px; } .masonry-item { grid-row-end: span var(--rows); /* Dynamic spanning */ } /* Individual item heights */ .item:nth-child(1) { grid-row-end: span 20; } .item:nth-child(2) { grid-row-end: span 30; } .item:nth-child(3) { grid-row-end: span 25; }

üîë Key Technique:

The masonry effect is achieved by setting grid-auto-rows to a small value (like 10px) and then making each item span a different number of rows using grid-row-end: span N. This creates the staggered, brick-like appearance.

/* Mobile First - Single Column */ .responsive-grid { display: grid; grid-template-areas: "header" "main" "sidebar" "ads" "footer"; grid-template-columns: 1fr; } /* Tablet and Desktop */ @media (min-width: 768px) { .responsive-grid { grid-template-areas: "header header header" "sidebar main ads" "footer footer footer"; grid-template-columns: 200px 1fr 150px; } } /* Large Screens */ @media (min-width: 1200px) { .responsive-grid { grid-template-columns: 250px 1fr 200px; } }
